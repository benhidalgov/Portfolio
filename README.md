# Portfolio Project

This is a beginner-level portfolio webpage project built using Django for the backend and React for the frontend. This project aims to showcase personal projects, skills, and experience.

## Table of Contents

- [Features](#features)
- [Technologies Used](#technologies-used)
- [Prerequisites](#prerequisites)
- [Usage](#usage)
- [Contributing](#contributing)
## Features

- **Resume** Use it to make a interactive resume for myself
- **Portfolio:** Display projects with descriptions and images.
- **Skills and Experience:** List of skills and professional experience.
- **Contact Form:** My social media, Github or Linkedin 

## Usage

Showcase personal projects, on the languages used mostly during my college path and personal ones.
-Leave information for recruiters for them to review some of my projects.
-Learning new things.

## Technologies Used

- **Backend:** Django 5.0.7, Django REST Framework
- **Frontend:** React 18.3.1, Material-UI
- **Database:** SQLite (default for development)

## Prerequisites

- You can install the requirements for this project with pip and the doccument requirements.txt with the command in your terminal: "pip install -r requirements.txt"
- Install node.js and the latest version of this one.
